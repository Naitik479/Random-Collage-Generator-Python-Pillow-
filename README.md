# Random-Collage-Generator-Python-Pillow-
A Python-based tool that automatically generates image collages from a folder of images. It randomly selects images without repetition, groups them, and places them into predefined positions on a template to create multiple collages efficiently.

📌 Random Collage Generator (Python + Pillow)

This project is a Python-based automation tool that creates collage images from a folder of images.

🔹 Features:
- Selects images randomly without repetition
- Groups images into sets of 6
- Places images into predefined coordinates on a template
- Generates multiple collages automatically
- Maintains aspect ratio while resizing images

🔹 How it works:
1. Reads all images from a given folder
2. Shuffles them to ensure randomness
3. Splits them into groups of 6
4. Places each image onto a template using fixed coordinates
5. Saves all generated collages to an output folder

🔹 Tech Used:
- Python
- Pillow (PIL)
- OS & Random modules

🔹 Use Cases:
- Social media content generation
- Photo collage automation
- Batch image processing
